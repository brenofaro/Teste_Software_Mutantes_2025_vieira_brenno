

### Clonar repositorio da urllib3

```bash
git clone https://github.com/urllib3/urllib3.git
```

### Criar ambiente virtual

```bash
python -m venv .venv
```

### Ativar ambiente virtual

Windows:

```bash
.\.venv\Scripts\activate
```

Linux:

```bash
source .venv/bin/activate
```
